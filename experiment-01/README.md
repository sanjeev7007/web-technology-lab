# Experiment No: 1  
## Creation of Image Map to Fix Hotspots  

### Aim  
To create a web page using image map to fix hotspots and display related information.

### Software Used  
- Notepad / VS Code  
- Web Browser  

### Description  
An image map is a graphical image with clickable areas called hotspots.  
Each hotspot is defined using coordinates that link to different web pages.

HTML elements used:
- `<img>` to display the image
- `<map>` to define the image map
- `<area>` to define clickable regions

### Files Included  
- indiamap.html  
- andhrapradesh.html  
- karnataka.html  
- India map image  

### Result  
The image map was successfully created and hotspots were fixed to navigate
to different state information pages.
